# Code of Conduct

## Be respectful

All participants in this project should communicate professionally and treat each other with
respect. Harassment, discrimination, or personal attacks are not tolerated.

## Expected behavior

- Be welcoming and constructive.
- Focus on the code and engineering goals.
- Give actionable, honest feedback.
- Respect different experience levels and help others grow.

## Not expected

- Derogatory comments
- Harassment or discrimination
- Repeatedly disrupting discussions
- Publishing private information without consent

## Enforcement

Contributions that violate this standard may be removed. Repeated or severe issues
may result in temporary or permanent removal from the project community.
